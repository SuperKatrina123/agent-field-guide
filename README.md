# Agent Field Guide

一套面向**有 Agent 基础的开发者**的主题式参考手册。

定位不是资料堆放，也不是框架文档镜像，而是聚焦认知边界、上下文工程、系统设计、可靠性与工程化的系统化知识库。想随时翻查某个主题，从这里找。

## 这里有什么

- `00-*`：导航页，解决"从哪里开始读"
- `01–17`：主线章节，按知识递进展开
- `99-*`：附录、术语表与参考入口

## 推荐阅读顺序

第一次进入，建议按顺序：

1. [00-课程首页](00-课程首页.md)
2. [00-学习路径](00-学习路径.md)
3. `01 → 17` 主线章节
4. [99-附录与术语表](99-附录与术语表.md)

最短主线路径：`01 → 02 → 03 → 04 → 05 → 08 → 10 → 11`

### 主线章节

- `01–03`：重建 Agent 认知边界，进入上下文工程总论
- `04–07`：上下文结构、装配、工具、状态与记忆
- `08–09`：规划、分解、执行控制，以及 Workflow / 多 Agent 协作
- `10–13`：可靠性、评测、安全、成本与扩展性
- `14–17`：模式库、案例、工作台与前沿专题

## 使用建议

- **想系统学**：先走主线，不要一开始就跳模式库和案例页
- **想解决手头问题**：先看 [00-问题索引](00-问题索引.md)，再回到对应章节补底层认知
- **想边学边做**：从 [16-模板、清单与工作台](16-模板、清单与工作台.md) 切入
- **想查术语**：直接看 [99-附录与术语表](99-附录与术语表.md)
